## 零、物理AI是什么？为什么现在火？

### 概念定义

**物理AI（Physical AI）** 由英伟达CEO黄仁勋在2024-2025年反复提出，核心含义：

```
传统AI（数字AI）          物理AI（Physical AI）
─────────────────    →    ─────────────────
ChatGPT、Midjourney        机器人、自动驾驶
在数字空间运行             在物理世界运行
输入文字 → 输出文字         感知环境 → 物理行动
不需要"身体"              需要"身体"（embodiment）
```

**一句话理解**：物理AI = 能看、能听、能理解物理世界、能在物理世界行动的AI系统。

### 为什么现在火？

| 驱动力 | 说明 |
|--------|------|
| **大模型突破** | GPT/多模态模型让机器真正"理解"物理场景，不再靠人工编程规则 |
| **英伟达推波助澜** | Omniverse + Isaac + Jetson 构建物理AI全栈平台，资本市场跟随 |
| **机器人产业链成熟** | 减速器/电机/传感器成本降至临界点，量产可行 |
| **制造业劳动力短缺** | 中国制造业工人10年减30%，机器替人从"可选"变"必选" |
| **自动驾驶逼近L4** | 端到端大模型重塑智驾，华为/小鹏/理想L4在即 |

## 三、关键子赛道深度对比

### 3.1 自动驾驶：物理AI最确定的万亿场景

```
物理AI在自动驾驶的应用：

感知层（"眼睛"）  →  决策层（"大脑"）  →  执行层（"手脚"）
    ↓                    ↓                    ↓
摄像头/雷达/LiDAR     域控制器/AI芯片      线控制动/转向
舜宇/速腾/禾赛        德赛/地平线/黑芝麻    伯特利/耐世特/拓普
```

| 环节 | A股龙头 | 港股龙头 | 确定性 | 爆发节奏 |

| 执行 | 伯特利、拓普集团 | 耐世特 | ⭐⭐⭐⭐ | 跟随智驾渗透 |

> **核心判断**：自动驾驶是物理AI中**业绩兑现度最高**的子赛道。德赛西威和地平线分别是A股和港股最纯的智驾标的。

## 五、核心结论与投资建议

### 5.1 物理AI投资优先级

```
第一优先级（现在可以研究/建仓）：
  ├── 德赛西威（智驾域控，确定性强）
  ├── 汇川技术（自动化龙头，稳健成长）
  ├── 舜宇光学（车载光学，港股稀缺标的）
  └── 地平线（智驾芯片，港股稀缺标的）

第二优先级（等信号确认后介入）：
  ├── 绿的谐波（等机器人出货拐点）
  ├── 埃斯顿（等机器人市占率突破）
  ├── 小米集团（等机器人业务进展）
  ├── 速腾聚创（等盈利拐点）
  └── 恒立液压（等丝杠定点函）

第三优先级（观察为主，不急于动手）：
  ├── 奥比中光（等Q2增速恢复，PE太贵）
  ├── 拓普/三花/贝斯特（等定点函）
  ├── 鸣志电器（等量产订单）
  └── 优必选（回避，业绩未兑现）

回避：
  ├── 纯概念炒作的人形机器人公司
  ├── 商汤（烧钱模式不可持续）
  └── 任何"沾边就涨"但无业绩的杂毛
```

### 5.4 最后的忠告

> **物理AI是真实趋势，不是概念炒作。** 但市场总是在"过度乐观"和"过度悲观"之间摇摆。现在要做的不是追概念，而是：
>
> 1. **选有主业利润支撑的公司**——即使物理AI故事推迟，也不会血本无归
> 2. **等信号，不等故事**——订单、定点函、出货量拐点，才是真信号
> 3. **区分ToB和ToC**——ToB是现在就要用的真需求，ToC人形是10年的远故事
> 4. **警惕"完美叙事+离谱估值"的组合**——奥比中光375x PE就是这个陷阱的典型

## 三、六大核心量能信号

### 信号 1：两次爆量拉升的位置对比 🔴

| | 第一次爆量 | 第二次爆量 |
|------|:------:|:------:|
| 日期 | 6/5 | 6/11 |
| 价格区间 | 47 → 56 元 | 55 → 61 元 |
| 主力净买 | +21.74 亿 | +21.36 亿 |
| 涨幅 | +7.07% | +9.50% |
| 量能效率 | 3.07 亿/百分点 | 2.25 亿/百分点 |

**量能效率在下降**：第一次爆量，每花 3.07 亿推升 1%；第二次只需 2.25 亿。看似"效率提升"，实则是**盘子变轻了——因为筹码已经从机构转移到了散户手中，散户不会主动砸盘，所以拉升更容易。但一旦有大资金出货，散户接不住。**

### 信号 2：缩量加速赶顶 🔴🔴

```
6/5   爆量 +21.74 亿  →  涨 +7.07%   (底部启动)
6/11  爆量 +21.36 亿  →  涨 +9.50%   (二次加速)
6/15  主力仅 +1.32 亿 →  涨 +7.44%   ← 缩量加速！
6/16  主力仅 +3.85 亿 →  涨 +9.10%   ← 缩量加速！
6/25  主力仅 +2.07 亿 →  涨 +2.95%   ← 缩量新高！
```

**缩量加速 = 最危险的赶顶信号。** 涨得越猛、量越小，说明买盘越虚。6/15 涨幅 7.44% 但主力净买仅 1.32 亿，根本推不动这个涨幅——真实情况是卖盘消失了（散户惜售），而不是买盘强。

# 十倍股跟踪系统

> 基于"渗透率跟踪 + 信号评分"的十倍股研究辅助工具

## 快速开始

```bash
cd stock-tracker

# 每天早上跑一次 —— 5 分钟了解今天该关注什么
python3 tracker.py daily

# 发现新信号时 —— 录入到系统中
python3 tracker.py update

# 每周/每月跑一次 —— 查看完整评估和推荐
python3 tracker.py review

# 财报季（1/4/7/10月）—— 扫描财报关键词
python3 tracker.py earnings
```

## 命令一览

| 命令 | 用途 | 建议频率 |
|------|------|---------|
| `daily` | 每日检查清单 | 每天 |
| `update` | 录入新信号 | 发现信号时 |
| `review` | 完整评估报告 | 每周/每月 |
| `earnings` | 财报关键词扫描 | 财报季 |
| `add` | 添加新标的 | 需要时 |
| `edit` | 编辑/删除标的 | 需要时 |

## 信号评分体系

每个信号有不同权重（1-5 分），得分越高表示正面信号越多：

| 分数 | 建议 |
|------|------|
| ≥ 20 且 S 级 | 🔥 强烈关注-买入候选 |
| ≥ 12 且 S 级 | ⭐ 重点研究-等待加仓 |
| ≥ 12（非 S 级） | 🔔 信号强烈-可考虑升至 S 级 |
| ≥ 6 | 📋 常规跟踪 |
| < 6 | ⏳ 等待信号 |

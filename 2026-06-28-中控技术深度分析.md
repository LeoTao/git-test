## 一、公司速写

中控技术（SUPCON）是中国 **流程工业自动化领域的绝对龙头**。

| 维度 | 概况 |
|------|------|
| 成立 | 1999 年（源自浙江大学） |
| 上市 | 2020 年科创板（688777），2023 年瑞士二次上市 |
| 总部 | 杭州滨江 |
| 定位 | 流程工业智能制造整体解决方案 |
| 使命 | "让工业更智能，让客户更成功" |
| 核心产品 | DCS / PLC / SIS / APC / MES / 工业AI |

### 核心业务：一个"卖铲子"的公司

中控的核心是给 **炼油厂、化工厂、发电厂、药厂** 这些流程工业企业，提供"大脑"（控制系统）和"管家"（运营优化）。

```
流程工业 = 石化 + 化工 + 电力 + 冶金 + 制药 + 造纸 + 食品饮料 ...
              ↓
        需要 DCS 控制系统来"开车"和"稳产"
              ↓
        中控的 DCS 国内市占率 ~35%（连续多年第一）
        超越霍尼韦尔、横河、西门子等外资
```

### 业务板块（按最新战略）

| 业务线 | 产品 | 2025 状态 |
|--------|------|:---:|
| **控制系统** | DCS / PLC / SIS 安全仪表 | 基本盘，增速放缓 |
| **工业软件** | APC 先进控制 / MES / 仿真 / 数字孪生 | 第二曲线 |
| **仪器仪表** | 变送器 / 调节阀 / 在线分析仪 | 国产替代中 |

> 最值得关注的战略转型：从"卖设备"到"卖会员"。PlantMembership 是工业界的 SaaS 订阅——客户按年付费获取持续优化服务。如果跑通，收入会更可预测。

| 净利率 | 12.59% | **5.68%** | 4.96% | 🔴 暴跌 |

| EPS | 1.42 | **0.56** | 0.10 | 🔴 腰斩 |

#### 利润崩塌的解剖

```
FY2024 → FY2025：发生了什么？

营收：91.39 亿 → 80.73 亿（-10.66 亿，-11.7%）
营业成本：60.44 亿 → 55.53 亿（-4.91 亿，比例匹配）

但费用不降反升：
  销售费用：7.94 亿 → 8.50 亿（+7%）  ← 收入降了还在加销售
  研发费用：9.78 亿 → 9.51 亿（-3%）   ← 刚性强，降不动
  管理费用：4.12 亿 → 4.07 亿（-1%）   ← 刚性强

结果：毛利降了 5.9 亿，费用只降了 0.3 亿
    → 营业利润从 12.00 亿 → 5.02 亿（-58%）
```

**核心问题**：中控的费用结构高度刚性——研发 10 亿+销售 8 亿，营收降 11% 时这些费用几乎降不下来。这就是硬件公司转型的典型"阵痛"。

#### 季度趋势——Q4 几乎零利润

| 季度 | FY2024 净利 | FY2025 净利 | 同比 |
|------|:------:|:------:|:---:|
| Q1 | 1.45 亿 | 1.20 亿 | -17% |
| Q2 | 3.71 亿 | 2.34 亿 | -37% |
| Q3 | 2.00 亿 | 0.78 亿 | -61% |
| Q4 | 4.00 亿 | **~0.10 亿** | **-98%** |

> 🔴 FY2025 Q4 净利润仅约 1000 万（全年 4.415 亿 - 前三季 4.316 亿）。一家年营收 80 亿的公司，Q4 赚 1000 万——这不是正常的季节波动，这是"大洗澡"或者在转型阵痛中硬扛费用。

## 三、成长股真伪鉴定

### 3.1 3-5 年增速趋势

| 年份 | 营收 | 增速 | 归母净利润 | 增速 |
|------|------|:---:|------|:---:|
| FY2022 | ~66 亿 | — | ~7.8 亿 | — |
| FY2023 | ~86 亿 | +30% | ~11.0 亿 | +41% |
| FY2024 | 91.39 亿 | +6% | 11.17 亿 | +1% |
| FY2025 | 80.73 亿 | **-12%** | 4.42 亿 | **-60%** |

```
营收轨迹：66亿 → 86亿(+30%) → 91亿(+6%) → 81亿(-12%)
利润轨迹：7.8亿 → 11亿(+41%) → 11亿(+1%) → 4.4亿(-60%)

这就是"成长故事破裂"的经典走势。
FY2023 还是 30% 营收增速 + 41% 利润增速，
FY2025 变成了 -12% 营收 + -60% 利润。
```

**判断：🔴 过去 2 年完全不符合成长股特征。** 营收 CAGR（FY2022-FY2025）≈ 7%，利润 CAGR ≈ -17%。这不是成长股，这是正在萎缩的公司。

## 四、物理AI视角：工业AI的最佳落地场景

### 4.1 中控与物理AI的关联

中控可能是A股中 **"工业AI"落地最真实** 的公司——因为它有真实的客户、真实的生产数据、真实的控制场景。

| 物理AI环节 | 中控的位置 | 关联度 | 现实状态 |
|-----------|----------|:---:|------|
| 工业AI控制 | TPT 时间序列大模型 + APC | ⭐⭐⭐⭐⭐ | 已有落地案例 |
| 数字孪生 | 流程工业仿真 | ⭐⭐⭐⭐ | 产品成熟 |
| 智能制造 | MES + 自动化全栈 | ⭐⭐⭐⭐⭐ | 核心业务 |
| 机器人 | 有限（流程工业用机器人少） | ⭐⭐ | 非核心 |

### 4.2 TPT 时间序列大模型——最被低估的工业AI

中控推出了 **TPT（Time-series Pre-trained Transformer）**——一个面向流程工业的时间序列大模型。

```
传统 DCS 控制：PID 算法，按固定规则调节
TPT 模型控制：学习海量历史运行数据，预测最优控制参数

意义：如果这个模型能在炼油/化工等场景稳定运行，
      它就是一个真正"在物理世界产生价值"的 AI。
```

> 但目前 TPT 仍在早期推广阶段，对收入的贡献几乎可以忽略。这是远期期权，不是近期催化剂。

### 4.3 PlantMembership——工业版的SaaS转型

中控在推 **PlantMembership 会员订阅制**——客户按年付费，获得持续的控制优化、设备健康管理等服务。

```
传统模式：卖一套 DCS 系统，一次性收 5000 万，后续每年收 10% 维保费
会员模式：客户按年付费，中控持续提供服务，每年收 1000-2000 万

好处：收入更可预测、客户粘性更强、毛利率更高
代价：短期收入确认变慢（分摊到多年），利润表难看
```

> FY2025 营收下滑的部分原因，可能就是这种收入确认模式的变化。如果真是这样，那营收下滑不是"业务萎缩"，而是"收入确认节奏变化"——需要从年报附注和管理层讨论中找证据。

## 六、CAN SLIM 七维筛查

| 维度 | 评分 | 评价 |
|------|:---:|------|
| **C** 当季收益 | 🔴 | Q1 净利 7468 万(-38%)，扣非 4568 万(-41%) |
| **A** 年度收益 | 🔴 | 3 年净利润 CAGR 为负，FY2025 暴跌 60% |
| **N** 新事物 | 🔥 | 工业AI大模型+PlantMembership订阅制+瑞士上市 |
| **S** 供需 | ⚠️ | 合同负债稳但应收暴增，供需信号矛盾 |
| **L** 领涨 | 🔥 | DCS 市占率 35%，流程工业自动化绝对龙头 |
| **I** 机构 | ✅ | 机构重仓，瑞士二次上市增加国际投资者 |
| **M** 大盘 | 🔴 | 市场震荡，不利于高估值标的 |

```
CAN SLIM 总分：2.5/7

C（当季收益）和 A（年度收益）全挂——欧奈尔最看重的两个指标都不及格。
N（新事物/工业AI）和 L（领涨/市占率第一）是仅有的亮点。
估值 213x PE，M（大盘方向）也不配合。

按照欧奈尔的体系，这是一只"不应该现在买入"的股票。
```

## 八、综合结论

### 8.1 一句话总结

> **中控技术是"好公司，坏时候"。** DCS 市占率 35% 的护城河无可争议，工业AI的落地场景最真实。但 FY2025 营收 -12%、利润 -60%、Q4 几乎零利润，应收账款周转恶化，PE 213x。当下不是在交易"好公司"，而是在交易"转型成功后"的完美预期——而这个转型，才刚刚开始。

### 8.2 多空对照

| 看多理由 | 看空理由 |
|---------|---------|
| ✅ DCS 市占率 35%，护城河真 | 🔴 FY2025 利润暴跌 60%，FY2026 Q1 继续 -38% |
| ✅ 合同负债 15.4 亿稳定，订单没崩 | 🔴 应收账款 38 亿周转恶化，回款差 |
| ✅ 工业AI（TPT大模型）落地场景最真 | 🔴 PE 213x，扣非 PE 284x |
| ✅ PlantMembership 转型SaaS模式 | 🔴 政府补贴+投资收益 > 净利润 |
| ✅ 现金 49 亿，资产负债率 45% | 🔴 季度利润从 4 亿(2024 Q4)骤降至 1000 万(2025 Q4) |
| ✅ 瑞士上市+国际投资者 | 🔴 存货 34 亿 + 长期投资 17 亿 |
| ✅ 回购 5 亿+，管理层认为低估 | ⚠️ 转型阵痛期可能持续 2-3 年 |

### 8.3 核心待验证问题

**中控到底是"转型阵痛"还是"业务萎缩"？**

```
转型阵痛论（乐观）：
  从卖 DCS → 卖 PlantMembership 会员
  收入从一次性确认 → 分 3-5 年确认
  短期收入下滑是会计现象，不是业务问题
  支撑：合同负债 15.4 亿稳定

业务萎缩论（悲观）：
  下游石化/化工客户资本开支减少
  DCS 渗透率见顶，竞争对手蚕食份额
  收入下滑是真实的竞争失利
  支撑：毛利率持续下行（33.9% → 31.0%）
```

> 这个问题的答案，需要看 FY2025 年报的**管理层讨论与分析**和**收入构成附注**才能判断。如果会员订阅收入在快速增长（如 +50%），那转型论成立；如果所有业务线都在萎缩，那萎缩论成立。

### 8.4 操作建议

```

  ├── 🔑 FY2026 中报（8月底）：营收是否恢复正增长？

  └── 🔑 TPT 大模型获得头部石化/化工客户付费订单

这些信号出现 3 个以上 → 转型论初步验证 → 可重新评估。
在此之前，中控属于"重点观察但暂不配置"的类型。
```
